# @duckwalk/mcp-server

`@duckwalk/mcp-server` is the public duckWalk MCP CLI for guided implementation sessions, PR review playback sessions, and Pathfinder codebase walkthroughs.

## Run

```bash
npx @duckwalk/mcp-server
```

After installation, the package also exposes:

```bash
duckwalk-mcp
```

## Tools

- `create_guided_session`
- `create_pr_review_session`
- `pathfinder`
- `get_guided_session`
- `update_step_status`
